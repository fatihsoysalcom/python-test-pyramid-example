# Python Test Pyramid Example

This example demonstrates the Test Pyramid concept in Python using a simple order processing system. It includes unit tests for individual components, integration tests for component interactions, and a simplified end-to-end test for a full workflow, illustrating how different test types contribute to software quality and confidence.

## Language

`python`

## How to Run

Save the code as `main.py` and run from your terminal:
`python main.py`
